As a coach I want to be able to create one to many coaching plans for a coachee. These should show as a list in target date order. Each coaching plan should be able to have a title, a description, a status (to do, in progress, done) and one to many actions as well as be assigned to a coachee which would be a list of coachees that I've added to the system. Each of the actions should be able to have a title, description, state (to do, in progress, done) as well as discussion where the coachee can @ a coach (also a list of people an administrator has added to the system). Coaching Plan's should also be able to have discussion as well. Actions for all in progress coaching plans should be visualised on a Kanban board with columns for the status, allowing the coachee to drag and drop actions between columns which should also update the status of the action.
